Firebase einrichten

Das Tippspiel braucht eine Firebase-Datenbank, damit alle Teilnehmer auf verschiedenen Geräten spielen können. Einmalige Einrichtung — ca. 5 Minuten.

1
Gehe zu console.firebase.google.com und melde dich mit Google an.
2
Klicke „Projekt erstellen", gib z.B. wm-tippspiel ein, Google Analytics kann deaktiviert bleiben.
3
Links im Menü → Build → Realtime Database → Datenbank erstellen → Im Testmodus starten → Fertigstellen.
4
Links → Projekteinstellungen (Zahnrad) → Allgemein → „Web-App hinzufügen" (</>). App-Nickname eingeben, dann siehst du den firebaseConfig-Block.
5
Öffne die Datei index.html im Texteditor, suche den Block unten und ersetze die Platzhalter mit deinen Werten:
const firebaseConfig = {
  apiKey: "DEIN_API_KEY",
  authDomain: "DEIN_PROJECT.firebaseapp.com",
  databaseURL: "https://DEIN_PROJECT-default-rtdb.firebaseio.com",
  projectId: "DEIN_PROJECT",
  storageBucket: "DEIN_PROJECT.appspot.com",
  messagingSenderId: "DEINE_SENDER_ID",
  appId: "DEINE_APP_ID"
};
6
Datei speichern → git add index.html && git commit -m "Firebase config" && git push → fertig!
